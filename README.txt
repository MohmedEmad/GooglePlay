1. INTRODUCTION
---------------
Project is automated search for a specific applications on Google play. This search will be by two different ways, select specific category and direct search from home page.


2. REQUIREMENTS
---------------
    * Visual Studio
    * Selenium 3.4
    * Chrome browser version 58.0.3029.110 (64-bit)
    * chromedriver.exe (version 2.27) 


3. INSTALLING & CONFIGUATIONS
-----------------------------
    * Open the project (GooglePlay.sln) in Visual Studio 
    * Open GooglePlay_Test
    * Run the first method (SelectApps_Test)
    * Wait and see the test result
    * Run the second method (SearchForFB_Test)
    * Wait and see the test result
OR
    * Open this path GooglePlay\Start\bin\Debug
    * Run start.bat
